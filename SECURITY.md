# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 3.8.x   | :white_check_mark: |
| 3.7.x   | :white_check_mark: |
| < 3.7   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in Power-Safety-UA, please report it responsibly.

**Do NOT open a public GitHub issue.**

Instead, please contact the maintainer directly:

- Email: **rekvizitor.ua@gmail.com**
- GitHub: [@weby-homelab](https://github.com/weby-homelab)

We will respond within 72 hours and work with you on a disclosure timeline.

### Scope

- API authentication bypass
- Token/secrets leakage
- Path traversal or file inclusion
- SSRF vulnerabilities
- Rate limiting bypasses
- Telegram webhook integrity

### Out of Scope

- Issues requiring physical access to the server
- Denial of Service (DoS) attacks without data compromise
- Self-hosted configuration issues
