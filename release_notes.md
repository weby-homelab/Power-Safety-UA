# Release v3.6.1

**PWA Dashboard with 24h AQI segments, 10-minute weather monitoring intervals, and Web Push Notifications**

## Що нового / What's New:
🇺🇦 **Українська:**
- **Годинні AQI стовпчики:** Збільшено кількість стовпчиків AQI на дашборді до 24 для кращого відображення добової динаміки.
- **Погодні метрики:** 12 годинних стовпчиків для температури та вологості на дашборді.
- **5-хвилинні інтервали збору:** Періодичність опитування Open-Meteo та інших API оптимізовано до 5 хвилин, а графіки малюються кожні 10 хвилин.
- **Web Push Notifications:** Додано підтримку веб-пуш повідомлень для сповіщення про зміну стану живлення у фоновому режимі (через VAPID).

🇬🇧 **English:**
- **Hourly AQI columns:** Increased the dashboard AQI columns count to 24 for better representation of daily dynamics.
- **Weather metrics:** 12 hourly columns for temperature and humidity on the dashboard.
- **5-minute collection intervals:** Poll frequency for Open-Meteo and other APIs optimized to 5 minutes, charts generated at 10-minute resolution.
- **Web Push Notifications:** Implemented service-worker Web Push notifications for power status updates (via VAPID).
